## PropFlow backend

### Requirements
- Node.js (18+ recommended)
- MongoDB (local or hosted)

### Setup
1) Create `.env` from `.env.example`
2) Install deps:

```bash
npm install
```

### Run

```bash
npm run dev
```

Health check: `GET /health`
Swagger UI: `GET /docs`
OpenAPI JSON: `GET /openapi.json`
